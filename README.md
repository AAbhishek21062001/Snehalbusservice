# 🚌 Snehal Bus Service

Snehal Bus Service is a school transportation platform designed to provide safe, reliable, and comfortable travel for students in Nagpur.

## ✨ Features

- 🏫 Partner Schools Section
- 🛣️ Bus Routes Information
- 🖼️ Gallery Section
- 📞 Contact Information
- 🔐 User Login & Registration
- 📱 Fully Responsive Design
- ⚡ Fast and User-Friendly Interface

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
- Node.js
- Express.js
- MongoDB
